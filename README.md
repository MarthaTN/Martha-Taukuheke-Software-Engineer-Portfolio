## 📌 About The Project

This project is a fully responsive and interactive portfolio website designed to:

- Showcase previous projects
- Allow clients to build and estimate their project cost
- Request consultations directly
- Contact me easily via a working email system

The design focuses on **clean UI, strong visual hierarchy, and user experience**.

---

## 🛠️ Built With

- HTML5
- CSS3 (Custom styling + modern UI techniques)
- JavaScript (DOM manipulation & logic)
- Bootstrap (Grid & responsiveness)
- EmailJS (Form handling without backend)

---

## ⚙️ Key Features

### 🎯 Interactive Project Builder
- Select project type
- Add features dynamically
- Real-time price calculation
- Live preview panel

### 📩 Consultation System
- Auto-fills selected project details
- Sends structured email via EmailJS
- Clean modal UI

### 📬 Contact Form
- Fully functional email system
- Real-time feedback messages

### 🖼️ Project Showcase
- Clickable project cards
- Modal gallery with images
- External project links

### 🎨 Modern UI/UX
- Glassmorphism effects
- Hover animations
- Responsive layout
- Consistent design system
